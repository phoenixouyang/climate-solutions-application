## About The Project
This is a web application that uses data from [Project Drawdown](https://drawdown.org/) to display information about their climate solution projects to share stories of innovation and hope! Please check out their website or the links on this page to learn more about the great work they're doing.

This page is written using Javascript, Ejs, and TailwindCSS, and features dynamic route handling, APIs from [DummyJSON](https://dummyjson.com/) and page error handling.
